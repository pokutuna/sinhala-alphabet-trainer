# 日本語 ↔ シンハラ文字 変換マッピング調査

`/convert` ページ(`app/lib/transliterate.ts`)で使う、日本語(かな・ローマ字)→ シンハラ文字のマッピング根拠と、現実装の検証結果をまとめる。

## 前提

- **公式の日本語 ↔ シンハラ変換規格は存在しない**。本マッピングは近似であり、判断には以下の証拠を使った。
  1. シンハラ語の音韻・正書法(英語版 Wikipedia, [r12a](https://r12a.github.io/scripts/sinh/si.html))
  2. **日本語固有名詞の実際の表記**(シンハラ語版 Wikipedia の記事名・リダイレクトを MediaWiki API で確認) — 最も決定力のある証拠
  3. 一般的な翻字慣習(Omniglot の Heļagana など)
- シンハラ文字は**アブギダ**。子音字が母音 a を内蔵し、母音記号(pilla)で他の母音を表す。
- 区別の用語: **歯音**(dental ත/ද/න)/ **反舌音**(retroflex そり舌 ට/ඩ/ණ)/ **無気音**(plain)/ **有気音**(aspirated, 息); **miśra** = サンスクリット・パーリ・外来語用の拡張字母。

## ユーザー指摘事項の詳細

### 促音(っ)は表現可能

シンハラ語は重子音(gemination)を持ち、**前の子音に hal kirīma /(virama)を付け、同じ子音を続ける**書き方で表す(r12a の例: අවුරුද්ද, කුරුල්ලා)。日本語の促音もこれで表現できる。

- 例: っか → `ක්ක`(k + virama + k)、北海道 = `හොක්කයිඩෝ` が実証。
- 実装方針: 促音トークンを単独 error にせず「**次の子音を取り込み、その頭子音を virama で重ねる**」処理にする(次トークン依存のため `tokensToSegments` 側で先読みが要る)。
- **注意(忠実でないケース)**: /h/(っは)・/ɸ/(っぷ→ෆ)・/ʃ/(っし→ෂ)の重子音はシンハラ語で native でない。これらは綴り上は書けるが音韻的には周辺的。実装では「重ねつつ note で注記」または当面 error 維持の判断余地あり。

### ちゃ/ちゅ/ちょ・しゃ・じゃ には専用の口蓋子音字がある

`sinhala.json` には既に口蓋音の子音字が揃っている。

| 行 | 専用字 | rom | IPA | data の kana |
| --- | --- | --- | --- | --- |
| ちゃ行 | ච | ca | tʃ | ちゃ |
| しゃ行 | ශ / ෂ | śa / ṣa | ʃ / ʂ | しゃ / しゃ(そり舌) |
| じゃ行 | ජ | ja | dʒ | じゃ |

現実装では ち/し を た行・さ行に押し込み、拗音 ちゃ を「た行 + yansaya(ත්‍ය = tya)」で作っている。これは **[tɕ] を [tʲ] 近似**しているのと同じで、専用字 ච を使う方が忠実。実例も ち = `චි`(高市 = තකඉచి)。

- 望ましい扱い:
  - **ち → චි(ca + i)**、**ちゃ/ちゅ/ちょ → ච + a/u/o**(`චා / චු / චෝ`)。yansaya 不要(子音が既に口蓋音)。
  - **し → ෂි or ශි**、**しゃ/しゅ/しょ → ෂ/ශ + 母音**。広島 = `හිරෝෂි`。
  - **じ → ජි**、**じゃ/じゅ/じょ → ජ + 母音**(現状の ja 経由でほぼ正しい)。
- 第1候補を口蓋子音、代替に従来の歯音/さ行近似を残すと、忠実さと選択肢を両立できる。

### 長音(母音連続)の長母音化

ヘボン式は**発音ベース**で長音を綴る(おう も おお も長音 ō)。日本語に二重母音はなく、えい も多くは長音 [eː](せんせい = se-N-se-e)。出典: [Romanization of Japanese — Wikipedia](https://en.wikipedia.org/wiki/Romanization_of_Japanese), [Hepburn romanization — Wikipedia](https://en.wikipedia.org/wiki/Hepburn_romanization), [sci.lang.japan](https://www.sljfaq.org/afaq/hepburn.html)。シンハラ文字には長母音記号(ā ී ū ේ ෝ)があり、長音をそれで表すのが適切。

母音連続を長母音化する段の対応:

| 直前の段 | 長音化する後続かな | 例 |
| --- | --- | --- |
| あ段 | あ | おかあさん |
| い段 | い | おにいさん |
| う段 | う | ふうせん |
| え段 | い, え | せんせい / おねえさん |
| お段 | う, お | とうきょう / おおさか |

**注意(誤爆)**: おう・えい は常に長音ではない(思う = おも・う、えいご = e-i-go、語境界の ない 等)。完全自動だと過剰修正になる。

**実装方針**: 母音連続は長母音化を**第1候補**に、畳まない素の連続(とう → `තො`+`උ`)を**代替候補**に出す。長音化は連鎖させない(とうう = とう + う)。明示的な長音記号 `ー`・マクロンは曖昧でないので代替なしで長母音に畳む。例: とうきょう → 第1 `තෝක්‍යෝ` / 代替で各 `තො`+`උ`。思う → `ඔ` `මෝ`(代替 `මොඋ`)。

## 実装状況(`app/lib/transliterate.ts` に反映済み)

1. ✅ **ふ/f**: 候補順を `["pa","fa"]` → `["fa","pa"]`(専用字 ෆ を第1)。
2. ✅ **ん**: 第1を `න්`(n+virama)、か/が行前は `ං`(anusvara)、ぱ/ば/ま行前は `ම්`。文脈で第1候補を切替。
3. ✅ **非音素候補の削除**: な行から `ṇa`、ら行から `ḷa` を除外。
4. ✅ **ち/し/じ の口蓋子音化**: ち→ච, し→ෂ を第1候補に(従来の歯音/さ行近似は代替で保持)。拗音 ちゃ/しゃ/じゃ は yansaya を使わず口蓋子音 + 母音で生成。
5. ✅ **促音っ**: 次子音を virama で重ねる(っか→ක්ක)。h/f/sh/ʃ の重子音は「重子音は近似」と note 付き。
6. ✅ **長音(母音連続)**: 母音連続を第1=長音化 / 代替=畳まない素に。`ー`・マクロンは確定で長母音化(マクロン入力 tō も子音に正しく結合)。

残課題(未対応・判断保留):

- **ざ行 z** はあいまい(ja か sa か定説なし)— 現状 ja 第1 / sa 代替の両候補。UI で「あいまい」と明示する余地。
- **つ tsu** は [ts] に相当字なし — 定説なし。
- **おう/えい の誤爆**は代替候補でユーザーが手動修正する設計(自動の語境界判定はしていない)。
